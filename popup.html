History browser

Yaq

Summarize or answer questions about any webpage.

Use cases:

  • Summarize blogs or news articles
  • Answer questions about product pages or YouTube videos
  • Inquire about GitHub issues or Linear tickets
  • Ask about currently open files in GitHub
  • Get assistance in reviewing GitHub PRs

Shortcuts:

  • Enter to answer
  • Shift + Enter to summarize
  • Ctrl/Cmd + Shift + M to open popup

Tips:

  • Customize buttons in plugin settings
  • Use Ctrl + number to trigger buttons
  • Experiment with different prompts
  • Use 'Continue conversation' for follow-ups
  • Select specific text for focused answers
  • YouTube responses use video subtitles
  • Browse history with 'History browser'
